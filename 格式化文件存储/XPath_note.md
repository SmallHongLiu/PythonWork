# XPath
- 在xml文件中查找信息的一套规则/语言，根据xml的元素或者属性进行遍历
- http://www.w3school.com.cn/xpath/index.asp
# XPath 开发工具
- 开源的XPath表达式编辑工具：XMLQuire
- Chrome插件：XPath Helper
- Firefox插件：XPath Checker

# 选取节点
- nodename: 选取此节点的所有的子节点
- /：表示从根节点开始选取
- //：表示选取节点，且不考虑位置
- . ：表示选取当前节点
- ..：表示当前节点的父亲节点
- @ ：表示选取属性

- XPath 中一般按照路径方法查找
